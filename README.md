# 📱 الماسح الضوئي - تطبيق Android حقيقي

تطبيق ماسح المستندات الكامل، مبني بـ **Capacitor** للحصول على وصول حقيقي لكاميرا الجوال وفلاش LED.

## 🎯 ما الفرق عن نسخة PWA؟

| الميزة | PWA (متصفح) | APK (هذا التطبيق) |
|---|---|---|
| فلاش الكاميرا الحقيقي ⚡ | ❌ مقيّد بالمتصفح | ✅ يعمل دائمًا |
| صوت غالق الكاميرا 🔊 | ✅ صناعي | ✅ صناعي |
| العمل بدون إنترنت | ✅ بعد الزيارة الأولى | ✅ من البداية |
| تثبيت بدون متصفح | ❌ | ✅ APK مباشرة |
| ظهور في درج التطبيقات | جزئيًا | ✅ تطبيق كامل |

## 🚀 الطريقة الأسهل: بناء APK مجانًا عبر GitHub (لا تحتاج تثبيت أي شيء)

### الخطوة 1️⃣: إنشاء حساب GitHub (إن لم يكن لديك)

ادخل [github.com](https://github.com/signup) وأنشئ حسابًا مجانيًا (دقيقتان).

### الخطوة 2️⃣: إنشاء مستودع جديد

1. اضغط على زر **"+"** أعلى يمين الصفحة → **New repository**
2. الاسم: `scanner-app` (أو أي اسم)
3. اجعله **Public** (أو Private)
4. **لا** تضع علامة على "Add a README"
5. اضغط **Create repository**

### الخطوة 3️⃣: رفع الملفات

أبسط طريقة (بدون أوامر):

1. في صفحة المستودع الجديد، اضغط الرابط **"uploading an existing file"**
2. اسحب جميع الملفات والمجلدات من هذا الـ ZIP إلى المتصفح
3. انتظر اكتمال الرفع (قد يستغرق دقيقة)
4. اضغط **"Commit changes"** في الأسفل

### الخطوة 4️⃣: انتظار البناء التلقائي

1. اذهب إلى تبويب **"Actions"** في أعلى المستودع
2. سترى عملية بناء قيد التشغيل (دائرة صفراء تدور 🟡)
3. انتظر 5-8 دقائق حتى تصبح علامة خضراء ✅

### الخطوة 5️⃣: تنزيل ملف APK

1. اضغط على عملية البناء المكتملة
2. في الأسفل ستجد قسم **"Artifacts"**
3. اضغط على **`scanner-app-apk`** للتنزيل
4. ستحصل على ملف ZIP يحتوي على `scanner-app.apk`

### الخطوة 6️⃣: تثبيت APK على جوالك

1. انقل ملف `scanner-app.apk` إلى جوالك (واتساب، Telegram، أو كابل USB)
2. افتح الملف على الجوال
3. سيطلب منك الجوال السماح بـ "تثبيت تطبيقات من مصادر مجهولة" → اقبل
4. اضغط **تثبيت**
5. ستجد التطبيق في درج التطبيقات بأيقونته وباسم "الماسح الضوئي"

🎉 **الآن الفلاش يعمل بضمان 100٪!**

## 🛠️ الطريقة الثانية: البناء على جهازك (للمطورين)

### المتطلبات

- [Node.js 20+](https://nodejs.org/)
- [Java JDK 21](https://adoptium.net/)
- [Android Studio](https://developer.android.com/studio) (يحوي Android SDK)

### الخطوات

```bash
# 1. تثبيت تبعيات npm
npm install

# 2. مزامنة Capacitor
npx cap sync android

# 3. فتح المشروع في Android Studio
npx cap open android

# 4. في Android Studio: Build → Build Bundle(s)/APK(s) → Build APK(s)
```

أو من سطر الأوامر مباشرة:

```bash
cd android
./gradlew assembleDebug
# ستجد APK في: android/app/build/outputs/apk/debug/app-debug.apk
```

## 🐛 حل المشكلات

### "تعذر تثبيت التطبيق" على الجوال
- تأكد من تفعيل "السماح بمصادر مجهولة" في إعدادات الجوال
- على Android 8+: عند الضغط على APK، سيطلب الإذن لتطبيق الملفات المستخدم (Chrome، WhatsApp...)

### عملية البناء فشلت في GitHub Actions
- افتح تفاصيل البناء وانظر السطر الأحمر
- المشاكل الشائعة: إصدار Java غير متوافق (الـ workflow يستخدم 21)
- جرب إعادة التشغيل من زر "Re-run jobs"

### الفلاش لا يعمل في APK
- تأكد من السماح للتطبيق بالكاميرا في إعدادات الجوال
- بعض أجهزة Huawei و Honor تحتاج إذن إضافي للفلاش
